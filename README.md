# Time_Yamyam
2 weeks FE projects - app for organizing study group

## 기획 의도
- 네카라쿠배과정을 겪으며 스터디 그룹이 많이 생성되었습니다. 이 때, 구두로 스터디 권유를 하는 것이 불편하다고 느껴 이 스터디 그룹 앱/웹을 만들기로 결정했습니다. 
- 포스팅을 통해 조원을 모집하고, 자신이 원하는 조건이 맞다면 신청할 수 있도록 기획했습니다. 

## 시간 먹는 하마 
인증미션 수행 여부가 기한마다 자동으로 확인되어 포인트를 부여하고, 이를 기반으로 유저의 성실도를 평가하여 참여와 모집 시 필터링할 수 있도록 한 스터디그룹 모집 어플리케이션.
유저와 스터디그룹, 게시글 등을 관리하는 CRUD를 구현하고, 서버통신 후 필요한 부분만을 업데이트하도록 AJAX와 MVC 패턴 사용.

## 사용한 기술
- Vanilla JS
- HTML
- SCSS
- Webpack
- Firebase
